## 1. 브랜드 포지셔닝 비교 (Krug vs Muse de Maree)

### 핵심 비교 매트릭스

| 요소 | Krug | Muse de Maree |
|------|------|---------------|
| **Brand Essence** | "The most generous expression of Champagne" -- 샴페인의 가장 관대한 표현. 인간의 장인정신이 만든 완성형 | "바다의 시간을 건네다" -- 인간이 만들지 않은 시간. 바다가 빚고, 우리는 그것을 기록할 뿐 |
| **Core Value Proposition** | 헤리티지(Heritage) -- 1843년부터 이어온 6대의 셀러 마스터, 축적된 전통의 무게 | 발견(Discovery) -- 깊이마다 다른 시간, 매번 새로운 관찰. 전통이 아닌 미지의 기록 |
| **Heritage Narrative** | "Since 1843" -- 시간의 선형적 축적. 과거에서 현재로 이어지는 연속성 강조 | "바다는 시간을 세지 않는다" -- 시간의 비선형적 경험. 깊이에 따라 다르게 흐르는 시간 |
| **Customer Relationship** | 감상자(Connoisseur) -- 브랜드가 완성한 것을 고객이 음미. 일방향 제공 | 관찰자(Observer) -- 브랜드와 고객이 함께 바다의 시간을 목격. 쌍방향 발견 |
| **Digital Experience 목적** | 제품 쇼케이스 + 브랜드 스토리텔링 -- 구매 동선을 우아하게 설계 | 디지털 아카이브 -- 구매 동선 자체가 없음. 기록과 관찰의 공간 |
| **Product Naming** | Edition (173eme Edition) -- 순차적 번호. 연속성과 개선의 서사 | Depth (첫 번째 깊이 ~ 왕의 깊이) -- 물리적 깊이. 공간과 압력의 서사 |
| **톤 앤 매너** | 절제된 자부심 -- 대문자, 정제된 어휘, 역사적 권위 | 담백한 고백 -- 소문자, 관찰 일기체, 자연에 대한 겸손 |
| **시각 은유** | 골드 + 크림 -- 따뜻한 빛, 셀러의 온기, 인간의 손길 | 틸 + 모래 -- 차가운 바닷속 빛, 해류의 움직임, 자연의 시간 |
| **경쟁 프레임** | "다른 샴페인 하우스보다 우월한 존재" | "비교 대상 자체가 없는 존재" -- 주류 브랜드가 아닌 아카이브 |

### 포지셔닝 전환의 핵심 논리

```
Krug의 서사 구조:
  인간(Joseph Krug) → 장인정신 → 완성된 제품 → 고객에게 전달

Muse de Maree의 서사 구조:
  자연(바다) → 시간과 압력 → 관찰된 결과물 → 함께 기록
```

Krug가 "우리가 만든 것"을 이야기한다면, Muse de Maree는 "우리가 발견한 것"을 이야기한다. 이 차이가 전체 브랜드 경험의 문법을 결정한다.

## 2. 브랜드 포지셔닝 전략

### "디지털 아카이브" 컨셉

Muse de Maree의 홈페이지는 주류 브랜드의 웹사이트가 아니다. 이것은 **바다의 시간을 보존하고 공유하는 디지털 기록 보존소**다.

**아카이브로서의 정체성:**
- 우리는 샴페인을 "판매"하지 않는다. 바다가 빚은 시간의 기록을 "공유"한다
- 사이트의 모든 콘텐츠는 "상품 정보"가 아닌 "관찰 기록"의 형식을 취한다
- 방문자는 "소비자"가 아니라 "관찰자"로 초대된다
- 제품 페이지는 "상세 스펙"이 아닌 "깊이별 관찰 일지"로 구성된다

### 왜 이 포지셔닝이 럭셔리에 유효한가

**1. 탈-상업의 역설 (The Anti-Commercial Paradox)**

럭셔리의 본질은 쉽게 가질 수 없는 것에 대한 갈망이다. 판매하지 않는 브랜드는 판매하는 브랜드보다 더 강한 욕망을 생성한다. Hermes가 대기 명단을 만들고, Patek Philippe가 "당신은 소유하는 것이 아니라 다음 세대를 위해 맡아두는 것"이라 말하는 것과 같은 원리다.

Muse de Maree는 이 원리를 극단까지 밀어붙인다: 사이트 어디에도 구매 버튼이 없다.

**2. 문화적 자본의 축적 (Cultural Capital Accumulation)**

디지털 아카이브 포지셔닝은 브랜드를 상업적 영역에서 문화적 영역으로 이동시킨다. 방문자가 경험하는 것은 "쇼핑"이 아니라 "탐험"이다. 이 경험 자체가 공유할 만한 문화적 가치를 갖는다.

**3. 시간의 비대칭성 (Temporal Asymmetry)**

Krug를 포함한 대부분의 럭셔리 브랜드는 "과거의 시간"(헤리티지)을 판다. Muse de Maree는 "자연의 시간"을 기록한다. 이것은 인간의 역사가 아닌 바다의 시간이기 때문에, 어떤 경쟁자도 복제할 수 없는 영역이다.

### "비상업적" 상업 전략 (The Non-Commercial Commercial Strategy)

```
일반적 럭셔리 퍼널:
  인지 → 관심 → 욕구 → 구매 → 충성

Muse de Maree 퍼널:
  호기심 → 관찰 → 몰입 → 기록(참여) → 소속감 → [구매는 브랜드 외부에서 자연 발생]
```

핵심 원칙:
- 홈페이지는 "구매 전환"이 아닌 "관계 전환"을 목표로 한다
- 구매 행위는 사이트 외부(프라이빗 채널, 오프라인, DM)에서 이루어진다
- 사이트 내에서 유일하게 허용되는 "전환"은 Ocean Circle 가입(관찰자 등록)이다
- 이 구조는 접근의 제한성을 높여 체감 가치를 극대화한다

### 감정적 영토 (Emotional Territory)

Muse de Maree가 소유하는 감정의 스펙트럼:

| 감정 | 설명 | 발현 지점 |
|------|------|-----------|
| **경이(Wonder)** | 바다 깊은 곳에서 일어나는 일에 대한 경탄 | Hero 영상, 깊이별 관찰 기록 |
| **발견(Discovery)** | 아직 알지 못하는 것을 처음 마주하는 설렘 | Archive 섹션, 새로운 Depth 공개 |
| **인내(Patience)** | 바다의 시간처럼 기다림의 가치를 이해하는 성숙함 | 숙성 기간 시각화, 대기 구조 |
| **자연의 시간(Nature's Time)** | 인간의 시간과 다른 리듬에 대한 인식과 존중 | 전체 사이트 톤, 관찰 일지 형식 |
| **소속(Belonging)** | 같은 것을 바라보는 사람들과의 조용한 연대 | Ocean Circle, 공유된 관찰 기록 |

## 3. 타겟 오디언스 페르소나

### 참고: Krug의 전형적 고객

Krug의 핵심 고객은 45-65세, 고소득 전문직, 와인 지식이 풍부하고 헤리티지 브랜드의 역사적 가치를 높이 사는 전통적 럭셔리 소비자다. 그들은 "이미 아는 것"을 확인하기 위해 브랜드를 방문한다.

Muse de Maree의 타겟은 다르다. 그들은 "아직 모르는 것"을 찾기 위해 온다.

|------|------|

**동기(Motivation):**
- 대량 생산되지 않은 것에 끌린다
- "이야기가 있는 것"을 소유하고 싶어한다
- SNS에 소비를 과시하지 않지만, 자신만의 아카이브(메모, 사진, 스크랩)를 꼼꼼히 관리한다
- 브랜드의 "태도"를 본다 -- 팔려고 하는 브랜드를 경멸한다

**디지털 행동 패턴:**
- Instagram: 팔로잉 300명 미만, 스토리보다 저장 기능 활용
- Pinterest: 비공개 보드 12개 운영
- 뉴스레터: 5개 이상 구독 (대부분 디자인/문화 관련)
- 검색: "해저 숙성 와인"이 아닌 "바다에서 숙성된 것들"로 검색

**사이트 이용 시나리오:**
서연은 어떤 매거진 기사에서 Muse de Maree를 발견한다. 사이트에 들어와 Hero 영상을 끝까지 본다. 첫 번째 깊이의 관찰 일지를 읽고, 3분간 머문다. Ocean Circle에 가입하지만 이름만 남기고 떠난다. 일주일 뒤, 새로운 관찰 기록이 올라왔다는 알림에 다시 방문한다. 그녀는 "구매"하러 온 적이 한 번도 없다.

**브랜드와의 관계:** 수집가 -- 기록을 모으고, 조용히 공유한다

**브랜드와의 관계:** 증인 -- 바다의 시간을 직접 목격하고, 그 기록을 공유한다

## 4. "판매 배제" 전략 (No-Sell Strategy)

### 왜 홈페이지에서 "구매"를 말하지 않는가

**원칙: 보이지 않는 것이 가장 강한 욕망을 만든다.**

Muse de Maree의 홈페이지에는 "구매하기", "장바구니", "가격", "주문" 같은 단어가 존재하지 않는다. 이것은 실수가 아니라 설계다.

**근거 1 -- 럭셔리의 역설**

럭셔리 마케팅의 가장 강력한 도구는 거절이다. Hermes의 Birkin은 "사고 싶어도 살 수 없는 것"이라는 구조가 욕망을 증폭시킨다. Muse de Maree는 이 원리를 디지털 경험에 적용한다. 사이트 자체가 "여기서는 살 수 없습니다"라고 말함으로써, 방문자는 "그러면 어디서?"라는 질문을 스스로에게 던지게 된다.

**근거 2 -- 아카이브의 정합성**

"디지털 아카이브"를 표방하면서 구매 버튼을 넣는 순간, 전체 서사가 무너진다. 박물관에서 전시품을 판매하지 않는 것과 같은 원리다. 기록 보존소의 경험은 상업적 동선과 양립할 수 없다.

**근거 3 -- 신뢰의 구조**

"팔지 않는 브랜드"는 "진짜인 브랜드"로 인식된다. 현대 소비자, 특히 밀레니얼과 Z세대는 상업적 의도를 즉시 감지하고 거부한다. 판매를 배제함으로써, 사이트의 모든 콘텐츠는 "광고"가 아닌 "기록"으로 읽히게 된다.

### 언어 전환 체계 (Vocabulary Transformation)

| 일반 커머스 언어 | Muse de Maree 언어 | 맥락 |
|-----------------|-------------------|------|
| 구매하기 (Buy) | 발견하기 (Discover) | CTA |
| 제품 (Product) | 기록 (Record) / 깊이 (Depth) | 카테고리 |
| 가격 (Price) | -- (표기하지 않음) | 생략 |
| 장바구니 (Cart) | -- (존재하지 않음) | 생략 |
| 리뷰 (Review) | 관찰 (Observation) | 사용자 콘텐츠 |
| 고객 (Customer) | 관찰자 (Observer) | 사용자 지칭 |
| 회원가입 (Sign Up) | 기록을 시작하다 (Begin Your Log) | 가입 |
| 베스트셀러 (Best Seller) | 가장 많이 관찰된 (Most Observed) | 인기 지표 |
| 신제품 (New Product) | 새로운 깊이 (New Depth) | 신규 라인 |
| 한정판 (Limited Edition) | 마지막 관찰 (Final Observation) | 한정 릴리즈 |

### 배제를 통한 욕망의 역설 (The Paradox of Desire Through Exclusion)

```
작동 메커니즘:

1단계: 부재의 인식
  방문자가 사이트를 탐색하며 "구매 버튼이 없다"는 사실을 인식한다.
  이것은 일반적인 사이트 경험과의 단절을 만든다.

2단계: 질문의 발생
  "살 수 없다면, 이것은 무엇인가?"
  "어떻게 하면 가질 수 있는가?"
  부재가 만든 공백을 방문자 스스로 채우려는 심리가 작동한다.

3단계: 가치의 재설정
  "판매하지 않는다 = 돈으로 살 수 없다 = 더 가치 있다"
  상업적 접근이 불가능할수록 체감 가치는 상승한다.

4단계: 관계의 전환
  구매 대신 "참여"라는 새로운 접근 경로를 발견한다.
  Ocean Circle 가입, 관찰 기록 공유, 커뮤니티 참여.
  이 과정에서 구매보다 더 깊은 브랜드 관계가 형성된다.

5단계: 자연 발생적 거래
  충분한 관계가 형성된 후, 프라이빗 채널을 통해
  "초대"의 형태로 구매 기회가 제공된다.
  이 시점에서 구매는 "거래"가 아닌 "선물"의 감각을 갖는다.
```

### 럭셔리 브랜드가 판매 없이 수요를 만드는 방법

**1. 콘텐츠가 제품을 대체한다**
사이트에서 소비되는 것은 샴페인이 아니라 "바다의 시간에 대한 기록"이다. 이 콘텐츠 자체가 충분히 가치 있어서, 구매 없이도 재방문의 이유가 된다.

**2. 접근 제한이 브랜드 가치를 증폭한다**
모든 것이 열려 있되, 유일하게 닫혀 있는 것이 "구매"다. 이 비대칭적 접근 구조가 브랜드의 독특한 위치를 강화한다.

**3. 커뮤니티가 판매 채널을 대체한다**
Ocean Circle 회원 간의 대화, 공유, 추천이 자연스러운 수요 생성 경로가 된다. 브랜드는 직접 판매하지 않지만, 커뮤니티가 그 역할을 자발적으로 수행한다.

**4. 희소성이 아닌 "부재"가 작동한다**
"100병 한정"(희소성)이 아니라, "여기서는 살 수 없습니다"(부재)가 작동한다. 희소성은 경쟁을 만들지만, 부재는 탐색을 만든다.

## 5. Ocean Circle vs Krug iD 비교

### Krug iD 개요

Krug iD는 병 뒷면의 6자리 코드를 스캔하여 해당 와인의 상세 정보에 접근하는 시스템이다.

**Krug iD 제공 정보:**
- 해당 Edition의 블렌딩 기록 (사용된 포도 품종과 빈티지 연도)
- Cellar Master의 테이스팅 노트
- 추천 음악 페어링 (Krug x Music 협업)
- 추천 푸드 페어링
- 보관 및 서빙 가이드

**Krug iD의 본질:** 제품 중심 -- "이 병에 대해 알아야 할 모든 것"

---

### Ocean Circle 개요

Ocean Circle은 사용자의 개인적 탐험 기록(Journey Log)이다. 특정 제품이 아닌, 사용자 자신의 발견과 관찰의 여정을 기록한다.

**Ocean Circle 제공 기능:**
- 탐험한 깊이의 기록 (어떤 Depth를 관찰했는가)
- 개인 관찰 일지 (자신만의 메모, 감각 기록)
- 다른 관찰자들의 기록 열람 및 공유
- 깊이별 발견 뱃지 시스템
- 시즌별 새로운 관찰 기록 알림
- 프라이빗 채널 초대 자격 (참여도 기반)

**Ocean Circle의 본질:** 사용자 중심 -- "바다의 시간 속에서 나는 무엇을 발견했는가"

---

### 상세 비교표

| 비교 항목 | Krug iD | Ocean Circle |
|-----------|---------|--------------|
| **접근 방식** | 제품 코드 스캔 (병 → 정보) | 계정 가입 (사람 → 여정) |
| **중심축** | 제품 (이 병의 이야기) | 사용자 (나의 발견 이야기) |
| **콘텐츠 생성자** | 브랜드 (Cellar Master 노트) | 브랜드 + 사용자 (공동 기록) |
| **시간의 방향** | 과거 (이 병이 어떻게 만들어졌는가) | 진행형 (나의 탐험은 계속된다) |
| **사용자 역할** | 정보 수신자 (읽는 사람) | 기록 참여자 (쓰는 사람) |
| **재방문 동기** | 새 병 구매 시 새 코드 스캔 | 새로운 관찰 기록, 커뮤니티 활동 |
| **사회적 기능** | 없음 (개인 경험) | 공유, 열람, 커뮤니티 |
| **게이미피케이션** | 없음 | 깊이 뱃지, 탐험 레벨 |
| **오프라인 연결** | 병 코드가 디지털 진입점 | 디지털 활동이 프라이빗 초대로 연결 |
| **데이터 구조** | 1:1 (한 코드 = 한 병 정보) | 1:N (한 계정 = 다수의 관찰 기록) |
| **브랜드 관계 모델** | 소비 기록 (Consumption Record) | 발견 일지 (Discovery Journal) |
| **최종 가치** | "이 샴페인을 더 깊이 이해한다" | "바다의 시간을 함께 기록한다" |

### 관계 모델의 근본적 차이

```
Krug iD:
  병 구매 → 코드 확인 → 정보 열람 → 소비 경험 강화
  (구매가 시작점, 제품이 중심)

Ocean Circle:
  사이트 방문 → 관찰 시작 → 기록 축적 → 관계 심화 → [초대]
  (호기심이 시작점, 사람이 중심)
```

Krug iD는 "이미 산 사람"을 위한 서비스다. Ocean Circle은 "아직 사지 않은 사람"을 위한 공간이다. 이 차이가 두 브랜드의 고객 관계를 근본적으로 다르게 만든다.

## 6. 전환 전략 (Conversion Framework)

### 핵심 CTA 언어 체계

사이트 전체에서 사용되는 CTA는 세 가지 동사 가족으로 구성된다.

**Primary CTA Family:**

| CTA | 한국어 | 사용 맥락 | 버튼 스타일 |
|-----|--------|-----------|------------|
| Discover | 발견하기 | 새로운 콘텐츠 진입, 깊이 탐색 | 투명 글래스, 넓은 패딩 |
| Record | 기록하기 | Ocean Circle 활동, 개인 관찰 메모 | 틸(#3D5A56) 배경, 소문자 |
| Share | 공유하기 | 관찰 기록 공유, 커뮤니티 전파 | 아웃라인, 투명 |

**절대 사용하지 않는 CTA:**
- 구매하기 / Buy / Purchase
- 장바구니에 담기 / Add to Cart
- 지금 주문 / Order Now
- 무료 체험 / Free Trial
- 할인 / Discount / Sale

### 전환 퍼널 설계

```
1. CURIOSITY (호기심)
   진입점: Hero 영상, SNS 유입, 매거진 기사
   목표: "이것은 무엇인가?"라는 질문 생성
   전환 지표: 스크롤 깊이 50% 이상, 영상 재생 완료율
   CTA: 없음 (자연스러운 스크롤 유도)

        ↓

2. DISCOVERY (발견)
   진입점: Observation 섹션, Archive 섹션
   목표: 콘텐츠 소비를 통한 브랜드 세계관 이해
   전환 지표: 페이지당 체류 시간 3분+, 2개 이상 깊이 탐색
   CTA: "다음 깊이를 발견하다" / "관찰 일지를 읽다"

        ↓

3. ENGAGEMENT (몰입)
   진입점: 특정 깊이의 상세 관찰 기록
   목표: 개인적 공감과 연결 형성
   전환 지표: 관찰 일지 전체 읽기, 다른 관찰자 기록 열람
   CTA: "나의 관찰을 시작하다" (Ocean Circle 가입)

        ↓

4. MEMBERSHIP (소속)
   진입점: Ocean Circle 가입
   목표: 지속적 참여와 기록 축적
   전환 지표: 월 2회 이상 방문, 1개 이상 관찰 메모 작성
   CTA: "기록하기" / "공유하기"

        ↓

5. ADVOCACY (전파)
   진입점: 축적된 관찰 기록, 커뮤니티 활동
   목표: 자발적 브랜드 전파, 프라이빗 채널 진입
   전환 지표: 외부 공유 횟수, 다른 관찰자와의 상호작용
   CTA: "이 관찰을 나누다"
```

### 마이크로 전환 포인트 (Micro-Conversion Points)

사이트 전체에 걸쳐 작은 전환 순간들이 분산 배치된다. 각 전환은 구매와 무관하며, 관계 심화를 위한 것이다.

**The Void (Home)**
| 전환 포인트 | 행동 | 측정 |
|------------|------|------|
| Hero 영상 몰입 | 영상을 끝까지 시청 | 재생 완료율 |
| 첫 스크롤 | Hero 아래로 자연 스크롤 | 스크롤 진입률 |
| Depth Teaser 클릭 | 특정 깊이 프리뷰 선택 | 클릭률 |

**Observation (Story)**
| 전환 포인트 | 행동 | 측정 |
|------------|------|------|
| 관찰 일지 열기 | 특정 깊이의 관찰 기록 진입 | 진입률 |
| 전체 읽기 | 관찰 일지를 끝까지 읽음 | 완독률 |
| 다른 깊이 비교 | 두 개 이상 깊이의 기록 비교 | 교차 탐색률 |

**Archive (Collection)**
| 전환 포인트 | 행동 | 측정 |
|------------|------|------|
| 깊이 카드 호버 | 제품 카드에 마우스 올림 | 호버율 |
| 상세 기록 진입 | 깊이별 상세 관찰 페이지 | 클릭률 |
| 5개 깊이 전체 탐색 | 모든 Depth를 한 번 이상 열람 | 완전 탐색률 |

**Ocean Circle (My Log)**
| 전환 포인트 | 행동 | 측정 |
|------------|------|------|
| 가입 | "기록을 시작하다" 클릭 | 가입 전환율 |

### 전환 언어의 설계 원칙

```
원칙 1: 동사는 항상 "발견"의 가족이다
  O: 발견하다, 관찰하다, 기록하다, 나누다, 시작하다
  X: 구매하다, 담다, 주문하다, 결제하다

원칙 2: 주어는 항상 "당신" 또는 "우리"다
  O: "당신의 관찰을 시작하세요" / "함께 기록합니다"
  X: "지금 구매하세요" / "놓치지 마세요"

원칙 3: 긴급성(Urgency)을 사용하지 않는다
  O: "바다의 시간은 기다립니다"
  X: "한정 수량!" / "오늘만 특가!"

원칙 4: 수량이나 가격을 언급하지 않는다
  O: "다섯 번째 깊이의 관찰이 시작되었습니다"
  X: "99,000원" / "100병 한정"
```

## 7. 메시지 프레임워크

### 브랜드 보이스 특성

**"담백한 고백" (Candid Confession)**

Muse de Maree의 보이스는 "고백"에 가깝다. 자랑하지 않고, 설득하지 않으며, 있는 그대로를 조용히 이야기한다. 마치 바다 앞에서 혼잣말을 하는 사람의 톤이다.

| 특성 | 설명 | 예시 |
|------|------|------|
| **절제 (Restraint)** | 필요한 만큼만 말한다. 수식어를 최소화한다 | "바다는 기다렸고, 우리는 열었습니다" (O) / "놀라운 해저 숙성 과정을 거친 환상적인 샴페인" (X) |
| **관찰자 시점 (Observer's Voice)** | 우리는 만든 사람이 아닌, 목격한 사람으로 말한다 | "15미터에서 우리가 발견한 것" (O) / "15미터에서 우리가 만든 것" (X) |
| **자연에 대한 겸손 (Humility to Nature)** | 바다가 주인이고, 우리는 기록자일 뿐이다 | "바다가 결정합니다. 우리는 기록합니다" (O) / "완벽한 바다를 선별합니다" (X) |
| **시적 간결함 (Poetic Brevity)** | 한 문장이 한 문단보다 강하다 | "깊이가 시간이 된 곳" (O) / "깊은 바닷속에서 오랜 시간 동안 숙성된 샴페인" (X) |
| **소문자의 자연스러움** | Krug의 대문자 권위 대신, 소문자의 편안함 | "discover the depth" (O) / "DISCOVER THE DEPTH" (X) |
| **한국어의 결 (Korean Grain)** | 과도한 영어 차용 없이, 한국어의 질감을 살린다 | "건네다", "빚다", "머물다" 같은 고유한 동사 사용 |

### 페이지별 핵심 메시지

**The Void (Home / Hero)**
```
Primary:    "우리는 그저 바다의 시간을 건넬 뿐입니다."
```

### 톤 가이드라인: "담백한 고백"의 실전 적용

**문장 구조:**
- 짧은 문장을 선호한다. 하나의 문장에 하나의 생각만 담는다
- 접속사를 최소화한다. "그리고", "하지만" 대신 마침표를 쓴다
- 수동태를 적극 활용한다. "바다에 의해 빚어진", "시간이 머문"

**피해야 할 표현:**
- 최상급 수식어: "최고의", "가장 뛰어난", "혁신적인"
- 상업적 긴급성: "지금", "놓치지 마세요", "한정"
- 기술적 자랑: "특허 받은", "세계 최초", "독보적인"
- 비교 표현: "~보다 우월한", "다른 어떤 것과도 다른"

**권장 표현 목록:**
| 용도 | 권장 표현 |
|------|-----------|
| 시간 표현 | 머무르다, 흐르다, 기다리다, 지나다 |
| 과정 표현 | 빚다, 건네다, 품다, 맡기다 |
| 발견 표현 | 마주하다, 열다, 기록하다, 관찰하다 |
| 깊이 표현 | 잠기다, 가라앉다, 닿다, 내려가다 |
| 감각 표현 | 고요한, 차가운, 느린, 깊은 |

### 메시지 위계 구조 (Message Hierarchy)

```
Level 1 - 브랜드 에센스 (모든 커뮤니케이션의 뿌리)
  "바다의 시간을 건네다"

    Level 2 - 포지셔닝 메시지 (왜 존재하는가)
      "바다가 빚은 시간의 기록을 보존하고 공유하는 디지털 아카이브"

        Level 3 - 가치 메시지 (무엇을 제공하는가)
          "깊이에 따라 다르게 흐른 시간의 관찰 기록"
          "함께 기록하고, 함께 발견하는 공간"
          "판매가 아닌 공유, 소비가 아닌 관찰"

            Level 4 - 행동 메시지 (무엇을 하라는가)
              "발견하기" / "기록하기" / "공유하기"

                Level 5 - 기능 메시지 (어떻게 작동하는가)
                  "Ocean Circle에서 나만의 관찰 일지를 시작하세요"
                  "다섯 개의 깊이를 탐색하세요"
                  "다른 관찰자의 기록을 열람하세요"
```

### 언어별 톤 분기

| 상황 | 한국어 톤 | 영어 톤 |
|------|----------|---------|
| Hero 카피 | 서정적, 고백체 | Minimal, lowercase |
| 관찰 일지 | 일기체, 관찰 보고서 톤 | Field notes, scientific calm |
| CTA | "~하다" 형 (발견하다, 기록하다) | Infinitive (discover, record) |
| Ocean Circle | 친밀한 2인칭 ("당신의") | Inclusive ("your", "our") |
| 오류/빈 상태 | 부드럽고 시적 | Gentle, poetic |
