**태그라인**: *"A place the maps don't show"*

## 3. 섹션 구조

### S0 — 히어로 (다크 포레스트)
- 숲 배경 일러스트 (어두운 그린 톤)
- 중앙에 골드 세리프체 "IRONHILL" 대형 타이포
- 위: "STEP ONTO THE" / 아래: "KNOWN TO SOME, FOUND BY FEW"
- 하단: "Scroll to explore" 유도 텍스트

### S1 — 브랜드 인트로 (라이트)
- 크림 배경으로 전환
- 대형 세리프 타이포: *"THERE'S A PLACE NOT FAR FROM HERE THE MAPS WON'T SHOW, KNOWN TO SOME AS THE IRONHILL."*
- 서브카피: "Beyond the noise, behind the veil, reached by foot and feel."
- 좌하: 위도 `25.3444° S` / 우하: 경도 `131.0369° E`
- **중앙을 관통하는 유기적 곡선 (지도 등고선 느낌)** — 어두운 선 + 연한 회색 선 2겹

### S2 — 스토리 + 이미지 콜라주
- 대형 타이포 유지 (연한 올리브)
- 우측에 본문 텍스트 (작은 산세리프)
- **빈티지 사진 콜라주**: 폴라로이드/필름 프레임 스타일
  - 기울어진 사진들
  - "00119_21", "00125_03" 등 필름 번호
  - "JOURNEY TO IRONHILL / 4/63 INR" 라벨
- 콜라주 사진들이 스크롤에 따라 패럴랙스로 이동

### S3 — 제품 카드 회전 (Keepers of Crafts)
- "MEET THE KEEPERS OF OUR CRAFTS" 대형 타이포
- 개별 제품 카드가 수동 스와이프/자동 회전 형태
- 각 카드:
  - **Warden of the Wild**: 위스키 잔 일러스트 + 빅풋 캐릭터
  - **Skunk Ape**: 홉 일러스트 + 유인원 캐릭터 (맥주)
  - **Mermaid's Tears**: 항아리 일러스트 + 인어 캐릭터
- 하단에 점 네비게이션 (dot indicator)
- "Explore" CTA 버튼 (다크 올리브 배경)

### S4 — 프로듀스 (제품 카드 덱)
- "IRONHILL PRODUCE" 라벨
- "SIMPLE WARES, BORN FROM HONEST WORK." 초대형 타이포 (배경 텍스트)
- **카드 덱 인터랙션**: 중앙에 제품 카드가 쌓여있고 스크롤/스와이프로 넘김
  - EGGS: 닭 일러스트 + 설명
  - BEEF: 소 일러스트 + 설명
  - HONEY: 벌 일러스트 + 설명
- 카드가 약간 기울어져 있어 자연스러운 느낌

### S5 — 푸터 (다크 그린)
- 거대한 "IRONHILL" 타이포 (크림색)
- 글자 사이에 캐릭터 일러스트 삽입 (빅풋, 인어 등)
- 이메일 구독 입력 필드
- 3컬럼 링크: Our Beverages / Our Produce / Beyond Ironhill
- 소셜: Instagram, LinkedIn
- 하단: Copyright, Privacy Policy, Terms of Service

## 4. 디자인 언어

### 타이포그래피
- **헤딩**: 클래식 세리프체 (대문자). 극도로 큰 사이즈 (화면 전체 너비)
- **본문**: 작은 산세리프. 우측 정렬 또는 우측 컬럼 배치
- **배경 타이포**: 연한 올리브색 초대형 텍스트가 배경 레이어 역할

### 이미지 처리
- 사진 대신 **동판화/보태니컬 스타일 일러스트** 사용
- 폴라로이드/필름 프레임 안에 이미지 배치
- 필름 번호 라벨링 ("00119_21", "JOURNEY TO IRONHILL / 4/63 INR")
- 기울어진 배치 + 패럴랙스 스크롤

### 헤더/내비게이션
- 투명 헤더, 좌측 로고(산세리프), 우측 햄버거(rounded rect 안에 `=`)
- 히어로에서는 숨김 → 스크롤 후 나타남
- 고정(sticky) 헤더

### 중앙 관통선 (Path Line)
- 페이지 전체를 관통하는 SVG 곡선
- 등고선/강줄기 느낌 — 어두운 실선 + 연한 회색선 2겹
- 스크롤에 따라 자연스럽게 이어짐
- 섹션 간 시각적 연결 장치

### 인터랙션
- 카드 덱 스와이프 (제품 카드)
- 제품 카드 회전/자동 전환
- 패럴랙스 스크롤 (이미지 콜라주)
- dot indicator 네비게이션

## 5. Muse de Marée 적용 가능 요소

### A. 페이지 관통 유기적 곡선 → 해류선/수심선
Ironhill의 등고선을 **해류선 또는 수심 등고선**으로 치환.
모든 섹션을 관통하며 스토리를 시각적으로 연결하는 장치.

### B. 빈티지 콜라주 이미지 처리
필름 프레임 + 패럴랙스 + 라벨링으로 아카이브/기록 느낌.
**Observation, The Maker 섹션**에서 활용 가능.
해저 촬영 이미지를 필름/다이브 로그 형태로 표현.

### C. 카드 덱 인터랙션 → Collection 섹션
제품을 카드 형태로 쌓아놓고 넘기는 경험.
**5개 큐베 라인업**을 카드 덱으로 표현 가능.

### D. 배경 대형 타이포
텍스트가 디자인 요소로 기능하면서 레이어 깊이감 생성.
연한 색 대형 텍스트 위에 콘텐츠/카드가 올라오는 구성.

### E. 좌표 표시
Ironhill: `25.3444° S / 131.0369° E`
우리: 해저 숙성 좌표 표시 가능 (한국 남해 좌표)
