# 히어로 섹션 — 최종안

**작성**: 럭셔리 브랜딩 총괄팀
**기준일**: 2026-02-22
**상태**: CEO 보고용 확정안

---

## 핵심 결론 (3줄 이내 요약)

히어로 섹션은 Phase 1(지상) → 수면 관통 전환 → Phase 2(심해)의 3단 구조로 확정한다. 카피는 `depth.` + 감각 3줄 최종안을 채택한다. 프론트엔드의 WebGL/글래스모피즘 스타일은 브랜드 기준에 불합치하므로 UX팀 방식(CSS clip-path + ScrollTrigger)으로 대체한다.

## 확정된 히어로 구조

```
[Phase 1 — 지상]
  배경: 코르크·라벨 엠보싱·병 디테일 클로즈업 정적 이미지 (필름 그레인, 앰버-크림 색온도)
  타이포: "depth." — Cormorant Garamond, 소문자, 중앙 정렬
  하위: "한국의 심해에서 숙성한 샴페인."
  스크롤 인디케이터: 가느다란 수직선 + 느린 펄스

         ↓ [수면 관통 전환 — 200vh 스크롤 구간]
  CSS clip-path + GSAP ScrollTrigger + SVG turbulence (UX팀 방식 채택)
  앰버(#EFDFBB) → 틸(#3D5A56) 색온도 전환
  역방향 스크롤 완전 지원

[Phase 2 — 심해]
  배경: 브랜딩 영상 루프 (기포 상승 → 빛 굴절 → 조류 흔들림, 15~20초)
  카피: 순차 페이드인 3줄 (아래 확정)
  CTA: "더 깊이 →" 텍스트 링크 (버튼 형태 금지)
```

## 팀별 산출물 검토 결과

### UX팀 (03) — 승인

콰이어트 럭셔리 원칙에 가장 충실하다. "강제하지 않는다. 압박하지 않는다"는 설계 철학이 브랜드 기준서 경험 영역과 완전히 일치한다. 수면 관통 전환의 세부 타임라인, 역방향 스크롤 지원, prefers-reduced-motion 대응까지 접근성 처리가 기준을 충족한다.

**수정 지시**: 없음. Phase 2 카피 타임라인은 카피팀 최종안 타이밍으로 교체한다.

---

### 프론트엔드팀 (01) — 조건부 승인 (핵심 수정 필요)

기술 역량은 우수하나 두 가지 브랜드 충돌이 있다.

**문제 1: 글래스모피즘(Glassmorphism) — 채택 불가**
`GlassmorphismOverlay` 컴포넌트는 `backdrop-blur`, `rounded-2xl`, 반투명 패널 등 테크 프로덕트 UI 문법을 따른다. 럭셔리 브랜딩에서 글래스모피즘은 과잉 인터랙션에 해당한다. 카피는 배경 영상 위에 직접 텍스트로 올린다. 오버레이 패널은 삭제한다.

**문제 2: 심해 배경 색상 토큰 — 수정 필요**
`rgba(3, 8, 28, 0.85)` (#03081c)는 브랜드 토큰 외 임의 색상이다. 심해 어두운 오버레이는 `#3D5A56` 계열의 딥 틸로 조정한다. 완전한 네이비-블랙 방향으로 가면 브랜드 팔레트를 벗어난다.

**문제 3: WebGL/R3F 수면 셰이더 — 재검토**
WebGL GLSL 셰이더는 기술적으로 정교하지만, 모바일 저사양 기기에서 CSS filter 폴백으로 자동 전환되는 구조가 이미 설계되어 있다. UX팀의 SVG turbulence 방식으로 통일하면 JS 번들 280KB를 절반 이하로 줄일 수 있다. Three.js + R3F 스택은 이 히어로 섹션에서 과잉 기술 투자다. 제거를 권고한다.

**유지 확정**: GSAP ScrollTrigger + Lenis 연동, Framer Motion 기포 파티클(기포 수 조정: 데스크톱 16개, 모바일 8개), 영상 lazy load IntersectionObserver.

### 카피팀 (04) — 승인

최종안 `depth.` + 감각 3줄이 럭셔리 언어 기준을 완벽하게 충족한다. 설명하지 않고, 주장하지 않고, 감각에 직접 말 건다. 대안 B(`still.`)의 역설 구조도 우수하나, 런치 시점 브랜드 인지도를 고려해 최종안을 유지한다.

**한 가지 수정**: Phase 2 카피 등장 순서 — UX팀 문서(카피 3줄 이후 CTA)와 카피팀 타이밍 가이드(9.5초 전체 페이드아웃 후 다음 섹션)가 불일치한다. 아래와 같이 통일한다.

```
Phase 2 sticky 진입 후:
  2.0s  — "수압이 먼저 닿는다." 페이드인
  4.5s  — "빛이 끊기는 깊이." 페이드인 (앞 줄 유지)
  7.0s  — "기포 하나가 위로 간다." 페이드인 (3줄 동시 노출)
  9.5s  — 3줄 유지 (페이드아웃 없음)
 11.0s  — "더 깊이 →" 페이드인 (CTA 텍스트 링크)
         이후 스크롤로 다음 섹션 진입
```

3줄을 페이드아웃하고 로고로 전환하는 원안은 채택하지 않는다. 카피는 잔류하고, 사용자가 스스로 스크롤로 떠난다.

---

### 영상팀 (06) — 승인

브랜드 원칙("제품을 보여주지 않는다", "선명함보다 질감", "시간의 무게")을 제대로 이해하고 있다. Scene A~D 구성, 색온도 가이드(7500K~8500K 쿨 딥 블루), 금지 사항 목록이 브랜드 기준서와 일치한다.

**확인 필요**: 프론트엔드 스펙(WebM < 8MB)과 영상팀 스펙(루프 1회 최대 30MB)이 최대 4배 차이난다. 웹 납품 기준을 20MB 이내로 합의해야 한다. 최종 타협점: WebM VP9 비트레이트 8Mbps, 루프 15초 기준 목표 15MB 이내.

---

### 마케팅팀 (05) — 히어로 섹션 직접 반영 항목 없음

마케팅 피드백의 핵심(데이터 아카이브, The Maker 섹션, Ocean Circle 멤버십, B2B 탭)은 히어로 이후 섹션에 적용될 사항이다. 히어로 섹션에 데이터나 신뢰 장치를 직접 삽입하면 "압박하지 않는다"는 경험 원칙을 훼손한다.

**히어로에서 반영하는 항목**: 스크롤 연동 압력 체험 효과 — UX팀의 수면 관통 전환 자체가 이 요구를 충족한다. 추가 애니메이션 레이어는 불필요하다.

**히어로에서 반영하지 않는 항목**: 한지 질감/수묵화 텍스처는 섹션 구분선 등 이후 섹션 디자인에 적용한다. 히어로 배경은 화석 산호 이미지와 심해 영상으로 충분하다.

## 럭셔리 기준 최종 검토

| 기준 영역 | 항목 | 판정 | 비고 |
|----------|------|------|------|
| **언어** | 어조 | 통과 | `depth.` — 속삭임. 선언 없음 |
| **언어** | 수식어 | 통과 | 형용사 없음. 감각 동사만 |
| **언어** | CTA | 통과 | "더 깊이 →" 텍스트 링크 |
| **언어** | 문장 밀도 | 통과 | 3줄. 각 줄 10자 이내 |
| **시각** | 컬러 | 검토 중 | 컬러 미확정 — 기획 과정에서 확정 예정 |
| **시각** | 타이포 | 통과 | Cormorant Garamond + Noto Sans KR |
| **시각** | 여백 | 통과 | 텍스트 최소화, 배경이 주역 |
| **시각** | 이미지 질감 | 통과 | 필름 그레인 필수, 코르크·라벨 엠보싱 촉각적 텍스처 |
| **시각** | 애니메이션 | 통과 | scrub 1.2, ease-out 기반, 급격한 모션 없음 |
| **경험** | 진입 | 통과 | 팝업·배너 없음. 3초 안에 브랜드 감도 인지 |
| **경험** | 탐색 | 통과 | 강제 전환 없음. 스크롤 자율 |
| **경험** | 제품 표현 | 통과 | 제품 직접 노출 없음. 감각으로만 존재 |
| **경험** | 마찰 | 통과 | 스크롤 인디케이터만. 알림·팝업 없음 |
| **기술** | 성능 | 통과 | LCP < 2.0s, WebGL 제거로 번들 경량화 |
| **기술** | 모바일 | 통과 | 동일 몰입감. 데스크톱 축소판 아님 |
| **기술** | 접근성 | 통과 | prefers-reduced-motion 완전 대응 |

**전체 기준 통과. 조건부 수정 사항(글래스모피즘 삭제, 색상 토큰 수정)은 프론트엔드팀 전달 완료.**

## 마케팅 피드백 반영 여부

| 피드백 항목 | 히어로 반영 | 처리 방향 |
|------------|------------|----------|
| 스크롤 압력 체험 효과 | 반영됨 | 수면 관통 전환이 압력 체험의 본질을 충족 |
| 한지 질감/수묵화 텍스처 | 미반영 | 이후 섹션 디바이더에 적용 (디자인팀 전달) |
| 데이터 아카이브 시각화 | 미반영 | 히어로 이후 독립 섹션으로 신설 |
| The Maker 섹션 | 미반영 | 히어로 이후 독립 섹션으로 신설 |
| Ocean Circle 멤버십 CTA | 미반영 | 페이지 하단 별도 섹션 |
| B2B 탭 | 미반영 | 페이지 최하단 분리 배치 |

히어로 섹션의 역할은 단 하나다: 브랜드의 감각적 세계로 끌어들이는 것. 신뢰 장치, 데이터, 멤버십은 히어로가 그 역할을 완수한 이후에 등장해야 한다.
