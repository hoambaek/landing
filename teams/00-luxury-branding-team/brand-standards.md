# Muse de Marée — 럭셔리 브랜딩 기준서

> 총괄팀. 모든 팀의 산출물은 이 기준을 통과해야 최종 승인된다.
> 브랜드 방향 → [`docs/project.md`](../docs/project.md)

---

## 브랜드 핵심 정의

**콰이어트 럭셔리 (Quiet Luxury)**
과시하지 않는다. 설명하지 않는다. 존재 자체로 말한다.

**아카이브 디스커버리 (Archive Discovery)**
판매가 아닌 발견. 기록하고 전달하는 디지털 아카이브.

이 두 축이 교차하는 지점이 뮤즈드마레의 자리다.

---

## 럭셔리 심사 기준 (Review Checklist)

각 팀의 산출물을 검토할 때 아래 기준으로 판단한다.

### 1. 언어 (Language)

| 기준 | 통과 | 불통 |
|------|------|------|
| 어조 | 낮고 고요하게 속삭인다 | 크게 외친다, 강조한다 |
| 수식어 | 없거나 최소 | "최고의", "세계 유일의", "놀라운" |
| CTA | "발견하기", "경험 시작하기" | "지금 구매", "한정 특가" |
| 문장 밀도 | 짧고 여백이 있다 | 설명이 길고 빽빽하다 |

### 2. 시각 (Visual)

| 기준 | 통과 | 불통 |
|------|------|------|
| 컬러 | Abyssal Amber 팔레트: 웜 블랙 `#0A0908` + 그레이지 `#E8E5E1` + 골드 `#CCAD7B` | 임의 색상, 과포화 색상, 블루/틸 계열 |
| 타이포 | Cormorant Garamond + Noto Sans KR | 장식적 폰트, 굵은 헤드라인 |
| 여백 | 넓고 의도적이다 | 빽빽하고 정보 과부하 |
| 이미지 질감 | 필름 그레인, 수중 빛, 촉각적 | 쨍한 디지털, 스튜디오 화보 |
| 애니메이션 | 느리고 유기적 (0.5~1.0s) | 빠르고 튀는 움직임 |

### 3. 경험 (Experience)

| 기준 | 통과 | 불통 |
|------|------|------|
| 진입 | 3초 안에 몰입 | 팝업, 배너, 즉각적 CTA |
| 탐색 | 자발적 발견을 유도 | 강제 유도, 넛지 과잉 |
| 제품 표현 | 유물처럼, 시간의 기록으로 | 상품처럼, 스펙 나열로 |
| 마찰 | 고요한 침잠감 | 과도한 인터랙션, 알림 |

### 4. 기술 (Technical)

| 기준 | 통과 | 불통 |
|------|------|------|
| 성능 | LCP 2.0s 이하, 부드러운 스크롤 | 버벅임, 레이아웃 시프트 |
| 모바일 | 동일한 몰입감 | 데스크톱 축소판 |
| 접근성 | `prefers-reduced-motion` 대응 | 모션 강제 |

## 팀별 승인 흐름

```
각 팀 산출물 완성
        ↓
00-luxury-branding-team 검토
        ↓
   기준 통과? ──No──→ 피드백 전달 → 팀 수정
        ↓ Yes
    최종 승인 → 다음 단계 진행
```

---

## 팀별 핵심 지침

| 팀 | 럭셔리 관점 핵심 요구사항 |
|----|--------------------------|
| `01-frontend-team` | 코드가 브랜드다. 버벅임 = 럭셔리 훼손. 성능은 타협 불가 |
| `02-design-team` | 더하지 말고 덜어낼 것. 여백이 메시지다 |
| `03-ux-team` | 압박하지 않는다. 사용자가 스스로 발견하게 설계한다 |
| `04-copywriting-team` | 한 문장이 열 문장보다 강하다. 담백하게 |
| `05-marketing-team` | 판매 언어 금지. 우리는 갤러리다, 마트가 아니다 |
| `06-creative-video-team` | 텍스트 없이 감각으로만 전달한다 |
| `07-creative-photo-team` | 완벽한 사진보다 시간이 느껴지는 사진 |
