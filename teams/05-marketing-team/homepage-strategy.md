## 1. 퍼널 아키텍처

### 1.1 퍼널 구조: AIDA 매핑

| 퍼널 단계 | 섹션 | 역할 | 핵심 지표 |
|-----------|------|------|-----------|
| **Awareness** (인지) | 1. The Void | 감각 충격. 브랜드 존재 각인 | 3초 이내 첫 스크롤 유도율 |
| **Interest** (관심) | 2. Observation | 브랜드 세계관 전달. "왜 바다인가" | 섹션 체류 시간 |
| **Interest → Desire** (관심 → 욕망) | 3. Data Archive | 공학적 신뢰 확보. "설계된 것이다" | 데이터 인터랙션 횟수 |
| **Desire** (욕망) | 4. The Maker | 만든 사람의 신뢰 = 제품의 신뢰 | 스크롤 완독률 |
| **Desire** (욕망) | 5. Archive | 제품 희소성 체감. 소유 욕구 자극 | 컬렉션 탐색 깊이 |
| **Action** (행동) | 6. Ocean Circle | 1차 전환: 초대제 멤버십 등록 | 전환율 |
| **Action** (행동) | 7. For Professionals | 2차 전환: B2B 문의 | 문의 건수 |

### 1.2 퍼널 흐름도

```
The Void (100% 유입)
    ↓ 예상 잔존율 85%
Observation (85%)
    ↓ 예상 잔존율 70%
Data Archive (60%)
    ↓ 예상 잔존율 80%
The Maker (48%)
    ↓ 예상 잔존율 85%
Archive (41%)
    ↓ 예상 잔존율 75%
Ocean Circle (31%)
    ↓ 전환율 목표 12-15%
멤버십 등록 완료 (3.7-4.6%)

For Professionals — 별도 진입 경로 (내비게이션 직접 접근)
```

### 1.3 드롭오프 포인트 및 완화 전략

| 드롭오프 위치 | 예상 이탈률 | 원인 | 완화 전략 |
|--------------|------------|------|-----------|
| The Void → Observation | 15% | 히어로 영상 로딩 지연, 목적 불분명 | LCP 2.0s 이하 보장. 스크롤 유도 마이크로 모션 |
| Observation → Data Archive | 30% | 스토리에 공감하지 못하는 방문자 이탈 | 시각적 전환 효과로 자연스러운 연결. 섹션 간 텍스처 디바이더 |
| Data Archive → The Maker | 20% | 데이터에 흥미 없는 감성 중심 방문자 | 데이터를 기하학 애니메이션으로 예술화. 체류 부담 최소화 (15초 이내 핵심 전달) |
| Archive → Ocean Circle | 25% | 제품 탐색 후 이탈 (정보 수집 완료) | Archive 하단에 "이 기록은 계속됩니다" 전환 카피. 자연스러운 초대 유도 |

## 2. 섹션별 메시지 전략

### 섹션 1: The Void (히어로)

| 항목 | 내용 |
|------|------|
| **핵심 메시지** | depth. — 단어 하나가 브랜드 전체를 말한다 |
| **신뢰 장치** | 몰입형 영상 품질 자체가 브랜드 역량의 증거 |
| **감정 트리거** | 감각 박탈 → 호기심. "차갑다. 아무것도 없다. 기포 하나 올라간다." |
| **방문자가 느껴야 할 것** | "이건 평범한 사이트가 아니다. 여기엔 무언가 있다." — 경외와 호기심 |
| **체류 목표** | 8-12초 |
| **스크롤 유도** | 히어로 하단 마이크로 모션 (기포 상승 방향과 일치) |

### 섹션 2: Observation (스토리)

| 항목 | 내용 |
|------|------|
| **핵심 메시지** | 바다는 기다린다. 우리는 기록한다. |
| **신뢰 장치** | 관찰 일기 형식 — 브랜드가 시간과 함께 만들어졌다는 서사 |
| **감정 트리거** | 시간의 무게. 인내와 경이의 교차 |
| **방문자가 느껴야 할 것** | "이 브랜드는 급하지 않다. 기다릴 줄 아는 사람들이 만들었다." — 존경 |
| **체류 목표** | 15-25초 |
| **카피 톤** | 관찰자의 시점. 3인칭. 짧은 문장. 여백이 카피의 일부 |

### 섹션 3: Data Archive (해저 데이터)

| 항목 | 내용 |
|------|------|
| **핵심 메시지** | 단순히 바다에 넣은 것이 아니다. 설계된 것이다. |
| **신뢰 장치** | 실제 해저 60m 수온/압력 로깅 데이터의 시각화. 측정과 기록의 증거 |
| **감정 트리거** | 정밀함에 대한 경외. "이 정도로 진지한 사람들이구나" |
| **방문자가 느껴야 할 것** | "감성만이 아니다. 과학이다. 공학이다." — 신뢰와 놀라움 |
| **체류 목표** | 10-20초 |
| **표현 원칙** | 원시 그래프 사용 금지. 미니멀 기하학 파동 애니메이션. 데이터가 아닌 감각으로 전달 |

### 섹션 4: The Maker (만든 이)

| 항목 | 내용 |
|------|------|
| **핵심 메시지** | 최고의 생산자를 찾고, 한국 바다에서 숙성한다. 이 과정에서 이 샴페인이 태어났다. |
| **신뢰 장치** | 셀렉션한 샴페인 생산자의 철학 + 한국 바다 숙성 팀의 전문성. 만든 이들의 이야기가 브랜드 내러티브 |
| **감정 트리거** | 만든 사람에 대한 신뢰가 제품에 대한 신뢰로 전이 |
| **방문자가 느껴야 할 것** | "이 사람이 만들었다면, 이 제품은 진짜다." — 인물 신뢰 |
| **체류 목표** | 12-18초 |
| **형식** | "만든 이의 편지" 또는 아카이브 형식. 자서전이 아닌 단편적 기록 |

### 섹션 5: Archive (컬렉션)

| 항목 | 내용 |
|------|------|
| **핵심 메시지** | 다섯 개의 깊이. 각각의 시간이 다른 이야기를 품는다. |
| **신뢰 장치** | Underwater Aging Certificate, 시리얼 QR 코드, 테이스팅 리추얼 가이드 |
| **감정 트리거** | 희소성 인지 + 소유 욕구. "이건 유물이지 상품이 아니다" |
| **방문자가 느껴야 할 것** | "갖고 싶다. 하지만 쉽게 살 수 있는 게 아니구나." — 갈망 |
| **체류 목표** | 20-40초 (5개 라인업 탐색) |
| **제품 표현** | 스펙 나열 금지. 유물처럼 제시. 시간의 기록으로 |

### 섹션 6: Ocean Circle (초대제 멤버십)

| 항목 | 내용 |
|------|------|
| **핵심 메시지** | 바다의 원에 들어오세요. |
| **신뢰 장치** | 초대제라는 형식 자체가 희소성의 증거 |
| **감정 트리거** | 소속감 + 배타성. "이 원 안에 들어가고 싶다" |
| **방문자가 느껴야 할 것** | "초대받았다. 이 기회를 놓치고 싶지 않다." — 특별함과 긴급성(조용한) |
| **체류 목표** | 전환까지 30-60초 |
| **언어 규칙** | "가입" 금지 → "초대". "구독" 금지 → "참여". "무료" 금지 |

### 섹션 7: For Professionals (B2B)

| 항목 | 내용 |
|------|------|
| **핵심 메시지** | 당신의 공간에 바다의 시간을 들이세요. |
| **신뢰 장치** | 디지털 세일즈 키트 제공. 전문 용어와 데이터 기반 접근 |
| **감정 트리거** | 비즈니스 기회 인지. "내 고객에게 이것을 제공하고 싶다" |
| **방문자가 느껴야 할 것** | "이건 우리 비즈니스에 맞는 제품이다." — 전문적 확신 |
| **체류 목표** | 문의 제출까지 90-120초 |
| **분리 원칙** | 메인 감성 흐름과 완전 분리. 내비게이션 또는 최하단에서만 접근 |

## 3. 전환 설계 — Ocean Circle

### 3.1 초대 메커니즘

**콘셉트**: 뉴스레터가 아니다. 초대제 멤버십이다.

```
방문자가 섹션 6 도달
    ↓
"바다의 원에 들어오세요" 카피 노출
    ↓
최소 정보 입력 (이메일 + 이름)
    ↓
"초대가 전달되었습니다" 확인 화면
    ↓
웰컴 이메일 발송 (24시간 이내)
    ↓
멤버 전용 콘텐츠 접근 코드 제공 (향후)
```

**핵심 원칙**:
- "가입하세요"를 절대 쓰지 않는다. "초대"와 "참여"만 사용
- 폼 주변에 혜택을 나열하지 않는다. 혜택은 암시한다
- 초대장 도착 확인 화면이 브랜드 경험의 연장이어야 한다

### 3.2 폼 필드 (최소화)

| 필드 | 필수 여부 | 이유 |
|------|----------|------|
| 이메일 | 필수 | 유일한 커뮤니케이션 채널 |
| 이름 (성) | 필수 | 개인화된 초대장 발송 |
| 관심 분야 | 선택 | "개인 소장" / "선물" / "비즈니스" — 세그먼트 분류용. 드롭다운 아님, 태그 선택 |

**폼 필드 수**: 최대 3개. 럭셔리 브랜드는 고객에게 많은 것을 요구하지 않는다.

### 3.3 혜택 제시 위계

혜택을 직접 나열하지 않는다. 분위기로 전달한다.

```
1순위 (암시): 런칭 전 우선 구매 기회
   → "첫 번째 기록에 함께하세요"

2순위 (암시): 제작 과정 아카이브 접근
   → "바다의 시간을 먼저 읽으세요"

3순위 (암시): 한정 에디션 우선 알림
   → 명시하지 않음. 멤버가 된 후 자연스럽게 발견
```

**금지 표현**: "혜택", "무료", "독점 할인", "지금 바로", "놓치지 마세요"

### 3.4 가입 후 경험

| 타이밍 | 경험 | 형식 |
|--------|------|------|
| 즉시 | "초대가 전달되었습니다" 확인 화면. 해저 비주얼 배경 | 페이지 전환 |
| 24시간 이내 | 웰컴 이메일. 브랜드 세계관 소개 + 아카이브 프리뷰 | 이메일 (HTML) |
| 1주 후 | 제작 과정 비하인드 콘텐츠 1편 | 이메일 |
| 2주 후 | CEO의 편지 (The Maker 확장 콘텐츠) | 이메일 |
| 런칭 시 | 우선 구매 초대 알림 | 이메일 + SMS (동의자) |

### 3.5 전환율 목표

| 지표 | 목표 | 산출 근거 |
|------|------|-----------|
| Ocean Circle 섹션 도달률 | 전체 방문자의 31% | 퍼널 잔존율 계산 기반 |
| 도달자 중 폼 노출 인지율 | 80% | 스크롤 내 자연 노출 |
| 폼 인지자 중 작성 시작률 | 25% | 럭셔리 브랜드 평균 (20-30%) |
| 작성 시작자 중 완료율 | 85% | 필드 3개 이하의 간결함 |
| **최종 전환율 (도달자 대비)** | **12-15%** | 0.80 x 0.25 x 0.85 = 17% (보수적 조정) |
| **최종 전환율 (전체 방문자 대비)** | **3.7-4.6%** | 도달률 31% x 전환율 12-15% |

**월간 목표 (트래픽 5,000명/월 가정)**:
- Ocean Circle 신규 멤버: 185-230명/월
- 6개월 누적 목표: 1,100-1,400명 (런칭 전 DB)

## 4. 전환 설계 — B2B (For Professionals)

### 4.1 타겟 페르소나

| 페르소나 | 직책 | 니즈 | 의사결정 기준 |
|----------|------|------|--------------|
| **파인다이닝 F&B 디렉터** | 음료 총괄, 소믈리에 | 와인 리스트 차별화, 스토리텔링 소재 | 제품 품질 + 브랜드 스토리 + 마진율 |
| **호텔 바 매니저** | 바 운영 책임자 | 시그니처 메뉴 개발, VIP 고객 경험 | 독점 공급 가능성 + 프레젠테이션 품질 |
| **독립 소믈리에/와인 바** | 오너 소믈리에 | 유니크한 큐레이션, 고객 대화 소재 | 희소성 + 테이스팅 퀄리티 + 가격대 |
| **수입사/유통사** | 바이어 | 포트폴리오 차별화, 신규 카테고리 | 시장 포지셔닝 + 수익성 + 공급 안정성 |

### 4.2 Inquiry 폼 설계

| 필드 | 필수 | 형식 |
|------|------|------|
| 성함 | 필수 | 텍스트 |
| 이메일 | 필수 | 이메일 |
| 업장명 / 소속 | 필수 | 텍스트 |
| 업종 | 필수 | 선택: 파인다이닝 / 호텔 바 / 와인 바 / 수입유통 / 기타 |
| 문의 내용 | 선택 | 텍스트 영역 (200자 제한) |

**폼 필드 수**: 5개. B2B는 자격 확인이 필요하므로 B2C보다 많되, 6개를 초과하지 않는다.

**폼 주변 카피**:
- 헤더: "파트너십 문의"
- 서브: "당신의 공간에 바다의 시간을 들이는 방법을 함께 설계합니다."
- 제출 버튼: "문의 전달하기"
- 확인: "문의가 전달되었습니다. 3영업일 이내 회신드리겠습니다."

### 4.3 디지털 세일즈 키트 구성

문의 폼 제출 후 자동 발송 또는 다운로드 제공.

| 구성 요소 | 포맷 | 내용 |
|-----------|------|------|
| **Brand Lookbook** | PDF (12-16p) | 브랜드 스토리, 제작 과정, 비주얼. 업장 테이블 위에 놓아도 어울리는 품질 |
| **Technical Analysis** | PDF (4-6p) | 해저 숙성 과정, 수온/압력 데이터 요약, 성분 분석서, 테이스팅 노트 |
| **B2B Terms Sheet** | PDF (2p) | 최소 주문 수량, 가격 구간(범위), 납품 조건, 독점 공급 옵션 |
| **Presentation Deck** | PDF (8-10p) | 업장 담당자가 내부 보고에 활용할 수 있는 프레젠테이션 |

**디자인 기준**: 모든 PDF는 브랜드 스탠다드 준수. Cormorant Garamond + Noto Sans KR. 여백 중심. 사진은 필름 그레인 질감.

### 4.4 리드 자격 분류

| 등급 | 기준 | 후속 조치 | 응답 시간 |
|------|------|-----------|-----------|

```
       전통 (역사)
```

### 7.3 가격 인식 전략 (프리런칭)

**원칙**: 가격을 공개하지 않는다. 가격이 보이면 상품이 된다.

| 전략 | 실행 방법 |
|------|-----------|
| **가격 비공개** | 홈페이지에 가격 정보 일절 노출하지 않음 |
| **희소성 시그널** | "5개의 깊이, 각 한정 수량" — 수량 구체적 명시 안 함 |
| **품질 앵커링** | Data Archive의 공학 데이터 + 인증서 시스템으로 프리미엄 인식 형성 |
| **동류 브랜드 연상** | 레퍼런스(Krug, Dom Perignon)를 디자인 언어로 은유적 연결 |
| **초대제 진입 장벽** | Ocean Circle 멤버만 가격 정보 접근 → "가격을 알 자격이 있는 사람" |
| **B2B 별도 단가** | For Professionals 문의 후 개별 견적 |

**기대 효과**: 방문자가 "이건 비쌀 것이다"를 스스로 결론 내리게 만든다. 브랜드가 직접 말하지 않는다.

### 7.4 "Engineered Art" 포지셔닝 증거 체계

홈페이지 전체를 관통하는 "설계된 예술품" 메시지의 섹션별 증거:

| 섹션 | "Engineering" 증거 | "Art" 증거 |
|------|-------------------|------------|
| The Void | 몰입형 기술 (WebGL, 영상 기술) | 감각 카피, 시네마틱 영상 |
| Observation | 관찰 기록의 체계성 | 관찰 일기 형식의 문학적 서사 |
| Data Archive | **수온/압력 실시간 로깅 데이터** | 기하학 파동 애니메이션으로 시각화 |
| The Maker | **생산자 셀렉션 전문성** | **한국 바다 숙성 팀 이야기** |
| Archive | Underwater Aging Certificate, QR 코드 | 유물로서의 제품 프레젠테이션 |

**하나의 문장으로**: 모든 섹션이 "이것은 설계되었고, 동시에 아름답다"를 반복한다. 반복하되 같은 말을 하지 않는다.
