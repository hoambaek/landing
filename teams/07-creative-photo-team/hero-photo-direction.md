## 핵심 지시사항

**첫 이미지가 브랜드의 첫 문장이다.**

방문자는 0.3초 안에 "여기 있어야 할 이유"를 판단한다.
우리가 보여줄 것은 오브제의 **표면** — 시간이 남긴 자국, 압력이 만든 구조.

---

## 오브제 우선순위

```
[1순위] 코르크 — 히어로 첫 화면 주인공
[2순위] 병 라벨 엠보싱 / 병 유리 곡선
[3순위] 화석 산호 — 서포팅, 맥락 제공용
[4순위] 석회 토양 — 배경/레이어용
```

## 1. 코르크 (Cork) — 히어로 메인 샷

### 연출 의도

샴페인을 아는 사람은 코르크를 본 순간 안다. 설명이 필요 없다.
동시에 코르크의 기공(孔) 구조는 화석 산호만큼 깊고 촉각적이다.
이 오브제 하나로 브랜드가 말하려는 모든 것 — 시간, 압력, 숙성 — 이 담긴다.

### 촬영 스펙

| 항목 | 내용 |
|------|------|
| **피사체** | 사용된 코르크 (사용 흔적 O, 와인 얼룩 X, 숙성의 시간만 남은 것) |
| **앵글** | 매크로 클로즈업 (1:1 이상 배율) |
| **조명** | 45도 단일 레이킹 라이트 — 기공 구조 입체감 극대화 |
| **배경** | 석회암 슬래브 또는 백악질 토양 |
| **포커스** | 기공 패턴 선예도 최우선. 배경 아웃포커스. |
| **납품 컷 수** | 최소 5컷 (다양한 각도, 다양한 조명 강도) |

### 세부 지시

- 코르크 전체가 프레임에 들어오지 않아도 된다. **표면의 단면이 화면을 채운다.**
- 압착 흔적(코르크가 병 입구에서 눌린 흔적)이 보이는 쪽을 주인공 면으로 선택.
- "오래됐다"는 느낌이 나야 한다. 새것 코르크 금지.

## 2. 병 라벨 엠보싱 / 병 유리 곡선

### 연출 의도

샴페인 병의 디테일은 럭셔리의 직접적 신호다.
전체를 보여주지 않는다 — 일부분만, 그것도 가장 물성이 강한 부분만.

### 촬영 스펙

#### 라벨 엠보싱

| 항목 | 내용 |
|------|------|
| **피사체** | 라벨 종이 질감, 엠보싱 양각, 인쇄 잉크 미세 돌기 |
| **앵글** | 극단적 측면 레이킹 라이트 (빛이 거의 수평으로 입사) |
| **조명** | 단일 텅스텐 포인트 라이트, 90도 이상 측면 |
| **배경** | 다크 슬레이트 — 라벨 지면이 부각되도록 |
| **금지** | 브랜드명·로고가 선명하게 프레임 내 노출되는 구도 |

#### 병 넥(Neck) 유리 곡선

| 항목 | 내용 |
|------|------|
| **피사체** | 병목 곡선, 유리 내부 기포·불순물, 빛의 굴절 패턴 |
| **앵글** | 측면 조명으로 유리 내부 빛 굴절 포착 |
| **조명** | 후방 광원 (Backlight) — 유리 투과광 활용 |
| **배경** | 검은 벨벳 또는 어두운 석회 배경 |

## 3. 화석 산호 (Fossil Coral) — 맥락 오브제

### 연출 의도

히어로 첫 화면의 단독 주인공으로 쓰지 않는다.
코르크·병 디테일 이후의 서브 레이어, 또는 사이트 내 다른 섹션에서 활용.
방문자에게 "무엇인지 모를 이미지"가 아니라 — 먼저 오브제로 브랜드를 인식시킨 후 화석을 맥락으로 쓴다.

### 촬영 스펙

| 항목 | 내용 |
|------|------|
| **피사체** | 화석 표면의 기공 구조, 석회화된 단면 |
| **앵글** | 매크로(1:1), 45도 측면 조명 |
| **배경** | 무처리 석회 토양 위. 받침대·인공 소품 금지 |
| **추가 컷** | 화석이 토양에 반쯤 파묻힌 상태 |

---

## 4. 공통 기술 스펙

| 항목 | 스펙 |
|------|------|
| **해상도** | 최소 5000px 장변 |
| **포맷** | RAW (CR3 / ARW) 원본 + TIFF 16bit + 웹용 JPEG 90% |
| **색공간** | Adobe RGB → sRGB (웹 변환) |
| **웹 출력** | 3840×2160px, WebP 변환 권장, 최대 350KB |
| **납품** | 오브제당 최소 3컷 (메인 1 + 서브 2), 촬영 후 5영업일 이내 |

## 5. 조명 & 무드 가이드

| 항목 | 가이드 | 금지 |
|------|--------|------|
| **광원** | 단일 자연광 또는 텅스텐 단일 소스 | 다중 소프트박스, 링 라이트 |
| **색온도** | 2700K~3200K (황금빛 텅스텐) | 5500K 이상 쿨 데이라이트 |
| **그레인** | ISO 800~1600 필름 시뮬레이션 또는 아날로그 촬영 | AI 노이즈 제거 |
| **채도** | 자연 채도 또는 -10~-20% 디새츄레이션 | 비비드 프리셋 |
| **컨트라스트** | 미드톤 중심 소프트 컨트라스트 | HDR, 딥 블랙 크러쉬 |

**필름 레퍼런스**: Kodak Portra 400 스캔본 — 빛이 오브제 위에 얹히는 게 아니라 **안으로 스며드는 느낌**.
