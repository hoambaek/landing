## 섹션 2. observation.

### 서브 헤드라인

```
바다는 기록한다.
```

### 본문 카피 — 관찰 일기

```
2024. 11. 03.
수온 3.2도. 조류가 멈추었다.
병이 바닥에 닿은 지 47일째.
표면의 진동이 여기까지 오지 않는다.


2025. 02. 18.
압력이 코르크를 누른다.
안쪽에서 기포가 느려진다.
시간이 액체에 스며드는 소리가 있다면
이런 소리일 것이다.


2025. 06. 01.
해류가 병을 0.3도 기울였다.
효모가 반응한다.
바다가 숙성을 결정한다. 우리가 아니다.


2025. 10. 27.
인양 전날.
병에 손을 대면 차갑다.
바다의 시간이 손끝에 전해진다.
```

### CTA

```
기록을 따라가다 →
```

### 선정 근거

바다를 1인칭 관찰자로 설정하되, 일기의 화자는 바다도 사람도 아닌 중간 시점. 날짜와 수온 같은 데이터를 섞어 관찰 기록의 객관성을 유지하면서 마지막 문장에서만 감각이 열린다. "바다가 숙성을 결정한다. 우리가 아니다."가 이 섹션의 핵심 — 브랜드가 자연에 주도권을 양보하는 태도. 일기 형식은 아카이브 디스커버리 축과 직결된다.

## 섹션 3. data archive.

### 서브 헤드라인

```
단순히 바다에 넣은 것이 아니다.
설계된 것이다.
```

### 본문 카피

```
수심 60m. 수온 1.8-3.4도.
압력 7.1기압.
365일 무진동 환경.

지상의 셀러가 만들 수 없는 조건을
바다가 만든다.
```

### 데이터 시각화 라벨

```
수온 — temperature (°C)
압력 — pressure (atm)
조류 속도 — current velocity (m/s)
숙성 기간 — aging period (days)
```

### CTA

```
데이터를 탐색하다 →
```

### 선정 근거

카피를 최소화하고 데이터가 말하게 한다. 서브 헤드라인 "설계된 것이다"가 이 섹션의 존재 이유를 한 문장으로 정의한다. 데이터 라벨은 한국어 먼저, 영문은 보조 표기. 숫자가 감각을 대신한다 — 1.8도라는 숫자가 "차가운 바다"보다 정확하고, 정확함이 신뢰를 만든다. 본문 마지막 "바다가 만든다"로 섹션 2의 태도를 반복 강화.

## 섹션 4. the maker.

### 카피 방향 (재작성 필요)

기존 카피는 CEO 개인 이력(GIST 공학, 프랑스 현대미술) 중심이었으나, 콘셉트 변경에 따라 재작성 필요.

**새 방향**: 셀렉션한 샴페인 생산자의 철학 + 한국 바다에서 숙성하는 팀의 이야기. "만든 이"가 한 개인이 아닌 생산자·팀·바다의 협업임을 보여준다.

### CTA

없음. 편지는 끝맺음으로 충분하다.

### 유지 원칙

담백한 편지 형식, 겸손과 자부심의 교차. CTA를 두지 않는 이유: 편지에 버튼을 다는 것은 품위를 해친다.

## 섹션 5. archive.

### 서브 헤드라인

```
다섯 개의 깊이. 다섯 개의 시간.
```

### 5-depth 컬렉션

```
— 20m   surface murmur.
         수면 가까이. 빛이 닿는 마지막 깊이.

— 30m   quiet pressure.
         빛이 사라지기 시작한다. 압력이 대화를 시작한다.

— 40m   blind current.
         조류만이 움직인다. 눈을 감아도 같다.

— 50m   still archive.
         모든 것이 느려진다. 시간이 퇴적된다.

— 60m   the bed.
         바닥이다. 여기서 기다린 것만이 올라간다.
```

### 인증서 카피 (underwater aging certificate)

```
이 병은 대한민국 해역 수심 ___m에서
___일간 숙성되었음을 증명합니다.

인양일: ____. __. __
시리얼 넘버: MDM-____-____
```

### QR 코드 안내

```
병에 부여된 고유 코드.
스캔하면 이 병이 잠들어 있던
깊이, 온도, 기간의 기록이 열린다.
```

### 테이스팅 리추얼

```
눈.
잔에 따르기 전, 병을 본다.
바다가 남긴 흔적이 표면에 있다.

코.
잔을 가까이 가져간다.
첫 향은 바다. 그다음은 시간.

입.
천천히. 첫 모금은 작게.
기포가 혀 위에서 사라지는 속도를 느낀다.

여운.
삼킨 뒤 기다린다.
깊이에서 온 것은 천천히 떠오른다.
```

### CTA

```
컬렉션을 열다 →
```

### 선정 근거

다섯 깊이의 이름은 영문 소문자로 통일. 각 이름이 그 깊이의 감각적 특징을 함축한다. `surface murmur`(수면의 속삭임)에서 `the bed`(바닥)까지 하강 구조. 인증서는 공적 문서의 톤을 유지하되 최소한의 정보만 노출. QR 설명은 기능이 아닌 서사로 풀었다 — "잠들어 있던"이라는 표현이 병을 생명체처럼 다룬다. 테이스팅 리추얼은 지시문이 아닌 시적 안내. 마침표로 끊어 호흡을 만든다.

## 섹션 6. ocean circle.

### 서브 헤드라인

```
바다의 원.
```

### 본문 카피

```
ocean circle은 가입이 아니다.
초대다.

런칭 이전, 첫 인양분에 대한
우선 구매의 기회가 열린다.

바다의 시간을 가장 먼저 만나는 자리.
수량이 정해져 있고, 자리도 정해져 있다.
```

### 초대 폼 라벨

```
이름.
연락처.
바다에 남길 한 문장. (선택)
```

### CTA

```
원에 들어오다 →
```

### 선정 근거

"가입이 아니다. 초대다." 두 문장이 이 섹션의 전부를 말한다. 멤버십이라는 단어를 쓰지 않는다. 혜택을 나열하지 않는다. "우선 구매의 기회"라는 표현으로 실질적 가치를 전달하되 판매 언어를 피한다. 폼의 마지막 필드 "바다에 남길 한 문장"은 기능적으로 불필요하지만, 이 필드가 ocean circle을 단순 DB 수집이 아닌 의례로 만든다. 선택 사항임을 명시해 부담을 없앤다.

```

```

```
— 성분 분석서

```

### CTA

```

```

### 선정 근거

B2B 섹션이지만 기업 브로슈어 톤을 피한다. "바다의 시간을 테이블 위에 놓는 일"이 서비스 제안을 시적으로 전환. 업종 구분을 폼에 넣어 리드 분류를 돕되, 라벨은 한 단어씩으로 절제. 세일즈 키트는 "포함 항목"을 대시로 나열해 깔끔하게 처리. 전체적으로 브랜드 보이스를 유지하면서 실무자가 필요한 정보에 도달하는 데 장벽이 없도록 설계.
